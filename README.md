# Eclipse Shader Voxy

Eclipse Shader Voxy is a customized Minecraft shaderpack focused on atmospheric visuals and enhanced voxel-style rendering.

This project improves the visual experience with:

* cinematic lighting
* smooth shadows
* atmospheric fog
* enhanced distant rendering
* optimized performance
* custom visual tweaks

The shaderpack is designed for players who want a more immersive and cinematic Minecraft experience while keeping stable performance.

## Compatibility

* Iris Shaders
* OptiFine
* Minecraft 1.21+
