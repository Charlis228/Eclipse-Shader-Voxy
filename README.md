# Eclipse Shader Voxy

Eclipse Shader Voxy is a customized Minecraft shaderpack focused on atmospheric visuals, cinematic lighting and enhanced voxel-style rendering.

The shaderpack improves the visual experience with:

* cinematic lighting
* smooth shadows
* atmospheric fog
* enhanced distant rendering
* optimized performance
* custom visual tweaks

Designed for players who want a more immersive Minecraft experience while keeping stable FPS and compatibility with modern shader loaders.
